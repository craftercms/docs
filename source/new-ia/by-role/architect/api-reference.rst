=============
API Reference
=============

CrafterCMS provides the following options for accessing content for content rich applications.

* JavaScript API
* GraphQL
* REST API
* FreeMarker
* Groovy
* Java

For more information on the above APIs, see :ref:`newIa-api-reference`
