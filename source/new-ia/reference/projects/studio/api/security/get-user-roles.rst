+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |

+----------------------------+-------------------------------------------------------------------+

+----------------------------+-------------------------------------------------------------------+
|| Required Role             ||                                                                  |

Parameters
----------

+---------------+-------------+---------------+--------------------------------------------------+
|| Name         || Type       || Required     || Description                                     |
