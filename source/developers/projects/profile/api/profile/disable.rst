===============
Disable Profile
===============

Disables a profile.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || POST                                                             |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/crafter-profile/api/1/profile/:id/disable``             |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+-------------------+-------------+---------------+---------------------------------------------------------------------------------------------------------------------------+
|| Name             || Type       || Required     || Description                                                                                                              |
+===================+=============+===============+===========================================================================================================================+
|| accessTokenId    || String     || |checkmark|  || The ID of the application access token                                                                                   |
+-------------------+-------------+---------------+---------------------------------------------------------------------------------------------------------------------------+
|| id               || String     || |checkmark|  || The profile's ID                                                                                                         |
+-------------------+-------------+---------------+---------------------------------------------------------------------------------------------------------------------------+
|| attributeToReturn|| String     ||              || The name of the attributes to return (don't specify to return all)                                                       |
+-------------------+-------------+---------------+---------------------------------------------------------------------------------------------------------------------------+

-------
Example
-------

^^^^^^^
Request
^^^^^^^

``POST .../api/1/profile/592715d4d4c650e226b03b62/disable?accessTokenId=e8f5170c-877b-416f-b70f-4b09772f8e2d``

^^^^^^^^
Response
^^^^^^^^

``Status 200 OK``

  {
    "username": "john.doe",

    "id": "592715d4d4c650e226b03b62"
