===============
Get All Targets
===============

Get all Crafter Deployer targets.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/target/get-all``                                        |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+

-------
Example
-------

^^^^^^^
Request
^^^^^^^

``POST .../api/1/target/get-all``

^^^^^^^^
Response
^^^^^^^^

``Status 200 OK``

.. code-block:: json

  [
    {
	  "target_id" : "mysite",
	  "load_date" : "2017-01-26T10:00:1.234",
	  "configuration_file" : "/opt/crafter/deployer/conf/mysite.yaml"
    },
    {
	  "target_id" : "yoursite",
	  "load_date" : "2017-01-01T11:00:5.678",
	  "configuration_file" : "/opt/crafter/deployer/conf/yoursite.yaml"
    }
  ]

---------
Responses
---------

+---------+------------------------------+-------------------------------------------------------+
|| Status || Location                    || Response Body                                        |
+=========+==============================+=======================================================+
|| 200    || ``.../target/get-all.json`` || See example above.                                   |
+---------+------------------------------+-------------------------------------------------------+
|| 500    ||                             || ``{ "message" : "Internal server error" }``          |
+---------+------------------------------+-------------------------------------------------------+
