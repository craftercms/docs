==============
Set Item State
==============

Set workflow state for item.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || POST                                                             |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/services/api/1/content/set-item-state.json``            |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || N/A                                                              |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+-------------------+-------------+---------------+--------------------------------------------------+
|| Name             || Type       || Required     || Description                                     |
+===================+=============+===============+==================================================+
|| site             || String     || |checkmark|  || Site to use                                     |
+-------------------+-------------+---------------+--------------------------------------------------+
|| path             || String     || |checkmark|  || Path to use                                     |
+-------------------+-------------+---------------+--------------------------------------------------+
|| state            || String     || |checkmark|  || State to set                                    |
+-------------------+-------------+---------------+--------------------------------------------------+
|| systemprocessing || Boolean    || |checkmark|  || Set system processing to true or false          |
+-------------------+-------------+---------------+--------------------------------------------------+

-------
Example
-------

^^^^^^^
Request
^^^^^^^

.. code-block:: guess

    GET ../api/1/services/api/1/content/set-item-state.json?site=mysite&path=/config/studio/administration/config-list.xml&state=EXISTING_UNEDITED_UNLOCKED&systemprocessing=false



^^^^^^^^
Response
^^^^^^^^

``Status 200 OK``

.. code-block:: json

    { "result" : "Success" }
