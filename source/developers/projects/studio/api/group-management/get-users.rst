===================
Get Users per Group
===================

Get all Crafter Studio users for a group with an optional range for pagination.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/services/api/1/group/users.json``                       |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || Admin, site member                                               |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+---------------+-------------+---------------+--------------------------------------------------+
|| Name         || Type       || Required     || Description                                     |
+===============+=============+===============+==================================================+
|| group_name   || String     || |checkmark|  || Group name to use                               |
+---------------+-------------+---------------+--------------------------------------------------+
|| site_id      || String     || |checkmark|  || Site to look under                              |
+---------------+-------------+---------------+--------------------------------------------------+
|| start        || Integer    ||              || Start offset                                    |
+---------------+-------------+---------------+--------------------------------------------------+
|| number       || Integer    ||              || Number of records to retrieve                   |
+---------------+-------------+---------------+--------------------------------------------------+

-------
Example
-------

.. code-block:: json

	GET .../api/1/services/api/1/group/users.json?group_name=mygroup&site_id=mysite

.. code-block:: json

  {
    "users" :
    [
      {
        "username" : "jane.doe",
        "first_name" : "Jane",
        "last_name" : "Doe",
        "email" : "jane@example.com",
      },
      {
        "username" : "joe.bloggs",
        "first_name" : "Joe",
        "last_name" : "Bloggs",
        "email" : "joe@example.com",
      }
    ]
  }

||        ||                                            ||                                                  |
||        ||                                            || *Status message will indicate which entity*      |

+---------+---------------------------------------------+---------------------------------------------------+
|| 500    ||                                            || ``{ "message" : "Internal server error" }``      |
+---------+---------------------------------------------+---------------------------------------------------+
