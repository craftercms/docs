===========
Get Version
===========

Returns the Crafter Engine JVM version details.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/monitoring/version``                                    |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+

-------
Example
-------

^^^^^^^
Request
^^^^^^^

``GET .../api/1/monitoring/version.json``

^^^^^^^^
Response
^^^^^^^^

``Status 200 OK``

  :linenos:

  {
    "name":"Sun Java System Application Server",
    "packageVersion":"1.1",
