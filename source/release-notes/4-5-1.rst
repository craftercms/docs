.. index:: CrafterCMS version 4.5.1 Release Notes

------------------------
CrafterCMS Version 4.5.1
------------------------

Released *June 18, 2026*

.. important::

    Please see :ref:`upgrade-notes-for-craftercms-4.5.1` for an important note and
    :ref:`Breaking changes <breaking-changes-4-x>` for breaking changes and other
    things to consider before upgrading

    For instructions on how to update your CrafterCMS install, please see :ref:`upgrading-craftercms`

^^^^^^^^^^^^
New Features
^^^^^^^^^^^^

* New :base_url:`Studio <_static/api/studio.html#tag/monitoring/operation/getSysInfo>`, :base_url:`Engine <_static/api/engine.html#tag/monitoring/operation/sysInfo>`, and :base_url:`Deployer <_static/api/engine.html#tag/monitoring/operation/sysInfo>`  monitoring APIs for retrieving system information

^^^^^^^^^^^^
Enhancements
^^^^^^^^^^^^
* `Improved Studio startup time <https://github.com/craftercms/craftercms/issues/8652>`__

^^^^^^^^^
Bug Fixes
^^^^^^^^^
* `Fixed issue with the context rebuild failing after trying to get a content item during init <https://github.com/craftercms/craftercms/issues/8734>`__

* `Fixed issue with Transcode Video form in dark mode <https://github.com/craftercms/craftercms/issues/8718>`__

* `Fixed issue with incorrect internal-name field extracted on save when embedded content data source is used <https://github.com/craftercms/craftercms/issues/5905>`__

* `Fixed issue with indexing components outside of /site/components <https://github.com/craftercms/craftercms/issues/8627>`__

* `Fixed issue with modeling tool for content type name <https://github.com/craftercms/craftercms/issues/8574>`__

* `Fixed issue with remote_repository table columns width <https://github.com/craftercms/craftercms/issues/8565>`__

* `Fixed issue with JavaScript that contains JSON with escaped quotes and double quotes <https://github.com/craftercms/craftercms/issues/8546>`__

* `Fixed issue with the publisher when it tries to publish a blob file in S3 that doesn't exist <https://github.com/craftercms/craftercms/issues/8367>`__

* Fixed issue with misconfigured blob store error message

* Fixed intermittent Maven Grab issue

* Fixed issue with git cleanup at startup when git status times out

^^^^^^^^^^^^^^^^^^
Dependency Updates
^^^^^^^^^^^^^^^^^^
* Noteworthy dependency updates:

  - Spring security to version 6.5.11
  - Spring core to version 6.2.19
  - Spring boot to version 3.5.14
  - Tomcat to version 11.0.20
  - OpenSearch to version 3.6
  - AWS SDK to version 2.46.10
  - Tika to version 3.3.1
  - Apache Log4j to version 2.26.0
  - BouncyCastle to version 1.84
  - Netty to version 4.2.12.final
  - Handlebars to version 4.6.0
  - Groovy to version 5

For a complete list of community tickets, please visit: `GitHub 4.5.1 <https://github.com/orgs/craftercms/projects/35/views/1>`_

For information on upgrading 3.1.x sites, see :ref:`here <compatibility-with-3.1.x>`
