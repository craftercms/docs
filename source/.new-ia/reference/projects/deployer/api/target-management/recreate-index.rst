==============
Recreate Index
==============

Recreate the underlying Elasticsearch index for a Crafter Deployer target.

--------------------
Resource Information
--------------------

.. include:: /includes/deployer-api-url-prefix.rst

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || POST                                                             |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/1/target/recreate/:env/:site_name``                       |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

+-------------------------+-------------+---------------+----------------------------------------+
|| Name                   || Type       || Required     || Description                           |
+=========================+=============+===============+========================================+
|| env                    || String     || |checkmark|  || The target's environment (e.g dev).   |
+-------------------------+-------------+---------------+----------------------------------------+
|| site_name              || String     || |checkmark|  || The target's site name (e.g mysite).  |
+-------------------------+-------------+---------------+----------------------------------------+
|| token                  || String     || |checkmark|  || The authorization token               |
+-------------------------+-------------+---------------+----------------------------------------+

-------
Example
-------

^^^^^^^
Request
^^^^^^^

``POST .../api/1/target/recreate/dev/mysite``

^^^^^^^^
Response
^^^^^^^^

``Status 202 OK``

---------
Responses
---------

+---------+----------------------------------+---------------------------------------------------+
|| Status || Location                        || Response Body                                    |
+=========+==================================+===================================================+
|| 202    ||                                 || See example above                                |
+---------+----------------------------------+---------------------------------------------------+
|| 404    ||                                 || ``{ "message" : "Target not found" }``           |
+---------+----------------------------------+---------------------------------------------------+
|| 500    ||                                 || ``{ "message" : "Internal server error" }``      |
+---------+----------------------------------+---------------------------------------------------+
